# Home Assistant 桌面客户端

基于 Electron 的 [Home Assistant](https://www.home-assistant.io/) 桌面应用（Windows / macOS / Linux）

![Home Assistant - Desktop](https://raw.githubusercontent.com/iprodanovbg/homeassistant-desktop/master/media/screenshot.png)

此项目 fork 自 [mrvnklm/homeassistant-desktop](https://github.com/mrvnklm/)

## 安装

直接从[发行页面](https://github.com/iprodanovbg/homeassistant-desktop/releases/latest)下载与你平台匹配的最新版本，即可安装 Home Assistant 桌面客户端。

## 使用 / 功能

- 悬停 / 单击托盘图标打开程序
- 支持多个 Home Assistant 实例（包括自动切换）
- 使用 bonjour 自动发现实例
- 右键上下文菜单，可设置/重置/退出应用
- 可启用全局快捷键（Cmd/Ctrl + Alt + X）用于显示/隐藏 Home Assistant
- 全屏模式（Cmd/Ctrl + Alt + Return）
- 自动更新（若上下文菜单未关闭此功能）

## 注意

- Windows 平台使用“分离窗口”模式时，不可直接拖动，需通过改变大小来移动窗口
