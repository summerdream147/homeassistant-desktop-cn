# Home Assistant Desktop - 汉化版

中文/[English](README_EN.md)

这是一个经过汉化和 Windows 端优化的 Home Assistant 桌面客户端。

## 功能特性

### 汉化内容
- 所有菜单项已汉化为中文
- 设置界面完全中文化
- 错误提示信息汉化
- 添加中文界面字体支持

### Windows 端优化
- 单实例锁支持（防止重复启动）
- 双击托盘图标快速打开窗口
- 托盘图标工具提示
- 优化窗口尺寸适配中文显示
- 支持任务栏固定
- 添加窗口边距避免贴边显示
- 优化深色主题显示效果

## 快捷键

- `Ctrl + Alt + X` - 显示/隐藏窗口
- `Ctrl + Alt + Enter` - 全屏模式

## 系统要求

- Windows 10/11
- Node.js 16 或更高版本

## 原版项目

本项目基于 [iprodanovbg/homeassistant-desktop](https://github.com/iprodanovbg/homeassistant-desktop) 进行汉化和优化。
